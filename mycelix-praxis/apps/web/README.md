# Praxis Web Client

React + TypeScript + Vite web client for Mycelix Praxis.

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Run tests
npm test

# Type check
npm run typecheck

# Lint
npm run lint

# Format
npm run format
```

## Features (Planned)

- Course discovery and enrollment
- Federated learning round participation
- Real-time round status tracking
- Credential issuance and verification
- DAO governance and voting
- Offline-first PWA support

## Architecture

- **React 18**: UI framework
- **TypeScript**: Type safety
- **Vite**: Fast build tooling
- **@holochain/client**: Holochain conductor communication
- **React Router**: Navigation
