# Next-Gen Learning Experience Plan

## The 5 Features (in build order)

### 1. Consciousness-Coupled UI ("The App Breathes")
- Wire 5 CSS variables from consciousness signals with 4-second EMA smoothing
- Stress → warmer colors, more spacing, slower animations
- Focus → chrome fades, content maximizes
- Motivation → accent saturation rises
- All transitions subliminal (4s ease-in-out)
- MVP: 2-3 days

### 2. Social Proof & Wise Feedback
- NSC examiner report data embedded ("68% of students struggled with this")
- Wise feedback error messages adapted by sovereignty mode
- Normalized struggle messages ("most students need 4-6 attempts")
- MVP: 2 days

### 3. Adaptive Game Difficulty
- Client-side BKT (Bayesian Knowledge Tracing) per concept
- Challenges adapt based on p_know estimate
- Consciousness engine modulates difficulty (+/-0.3 based on flow/warmup state)
- 5 difficulty tiers: Recognition → Guided → Target matching → Inverse reasoning → Transfer
- MVP: 3 days

### 4. Living Knowledge Graph (Constellation)
- Replace flat card grid with SVG constellation
- Force-directed layout clustered by subject, ringed by grade
- Mastered nodes glow, studying nodes pulse, unlockable nodes shimmer
- "Unlock burst" animation when prerequisites met
- MVP: 3-4 days

### 5. Session Orchestrator
- 25-minute structured sessions: warmup → new content → practice → interleaved review → cool-down
- Consciousness engine modulates pacing (extend during flow, break during stress)
- Session history persistence
- MVP: 3-4 days

## Total MVP: ~14-16 working days
## WASM size impact: ~24KB (negligible)
