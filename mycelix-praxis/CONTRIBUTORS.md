# Contributors

Thank you to everyone who has contributed to Mycelix Praxis! This project exists because of people like you.

---

## How to Get Added

Contributions of all kinds are welcome! You'll be added to this list when your first contribution is merged.

**Ways to contribute:**
- 💻 **Code**: Bug fixes, features, refactoring, tests
- 📝 **Documentation**: README improvements, guides, examples, tutorials
- 🎨 **Design**: UI/UX improvements, logos, diagrams, wireframes
- 🧪 **Testing**: Bug reports, test coverage, QA
- 💡 **Ideas**: Feature requests, architecture proposals, RFC discussions
- 📢 **Community**: Answering questions, mentoring newcomers, spreading the word

| 💡 Ideas | Accepted RFCs, feature designs | Listed in CONTRIBUTORS.md |
| 📢 Community | Answering issues, Discord help | Listed in CONTRIBUTORS.md |

---

## Recognition Criteria

To be added to this list, you must have:
1. Had at least **one contribution merged** into the main branch, OR
2. Provided **significant** help in discussions, issues, or community channels
3. Followed our [Code of Conduct](CODE_OF_CONDUCT.md)

**Note**: We manually update this file. If your contribution was merged but you're not listed, please:
1. Wait 1-2 weeks (we batch updates)
2. If still not listed, open an issue or comment on your merged PR
