    Class OfflineQueue<T>

    Offline-first operation queue with persistence.

    Queues operations when offline and syncs when reconnected. Supports priority ordering, retries, and persistence.

    Example

    const queue = new OfflineQueue({
      maxSize: 1000,
      maxRetries: 3,
      retryDelayMs: 5000,
    });

    // Queue operations when offline
    queue.enqueue('updateReputation', { agentId: 'a1', delta: 1 }, 1);

    // Process when online
    await queue.process(async (op) => {
      await api.execute(op.type, op.payload);
      return true;
    });
