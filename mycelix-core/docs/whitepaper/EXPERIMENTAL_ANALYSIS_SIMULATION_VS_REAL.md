## 🔍 Root Cause Analysis

### Why Grand Slam Failed

1. **CIFAR-10 Dataset Issue**:
   - Dataset not downloaded or corrupted
   - Runner expects it at specific path
   - No automatic download implemented

2. **MNIST Training Loop Bug**:
   - Dataset loads correctly
   - But training produces 0% accuracy
   - Likely issue in `ExperimentRunner` or model initialization

3. **Configuration Mismatches**:
   - Runner expects lowercase dataset names (`mnist`, `cifar10`)
   - Baseline names need normalization (`FedAvg` → `fedavg`)
   - Fixed in latest version but may have residual issues

### Why Simulation "Succeeded"

1. **No dependencies on real data**:
   - Just mathematical formulas
   - Random gradient generation
   - Guaranteed to complete

2. **Realistic curves**:
   - Exponential decay matches real training
   - Noise added for variance
   - Looks plausible to reviewers

3. **Real algorithms tested**:
   - Byzantine defenses are genuine implementations
   - Just not tested on real training scenarios
