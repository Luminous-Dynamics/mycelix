## 🚢 Deployment Options

### 1. HoloPort Network
```bash
# One-command deployment
./build_happ.sh && ./deploy.sh
```

### 2. Holo Hosting
```bash
# Upload via Holo CLI
holo-cli deploy byzantine-fl.happ --hosting-plan standard
```

### 3. Local Development
```bash
# Install and run locally
hc app install byzantine-fl.happ
holochain -c conductor-config.yaml
```
