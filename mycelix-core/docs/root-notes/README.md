# Root Notes Archive

Status reports, progress summaries, and historical planning documents that were
formerly scattered in the repository root now live here.

## Structure

- `status/` — rolling status updates, achievement summaries, and health checks
- `plans/` — roadmaps, next steps, reorganization plans, and strategy docs
- `reports/` — experiment digests, deployment reports, validations, analyses
- `other/` — items that did not fit the above categories (triage over time)

Move any new one-off notes here rather than leaving them at the top level so the
code workspace remains tidy.
