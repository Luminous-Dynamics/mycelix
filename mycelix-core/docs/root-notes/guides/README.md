# Root Notes Archive

Status reports, progress summaries, and historical planning documents that were
formerly scattered in the repository root now live here. They are intentionally
left unmodified so links inside the files still work.

Suggested subsections:
- `status/` – rolling status updates (PHASE, FINAL, SUMMARY files)
- `plans/` – roadmaps, next-step proposals
- `reports/` – experiment and deployment digests

Feel free to create subfolders as you continue curating documentation.
