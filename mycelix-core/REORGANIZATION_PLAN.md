## Current Architecture Understanding

### mycelix-workspace (The Unified Workspace)
- Location: `/srv/luminous-dynamics/mycelix-workspace/`
- Purpose: Unified monorepo workspace that links hApps via symlinks
- `happs/core` -> symlink to `../../Mycelix-Core`
- Contains: SDK (Rust), CLI, Observatory, shared components

### Mycelix-Core (This Repo)
- Location: `/srv/luminous-dynamics/Mycelix-Core/`
- Purpose: ZeroTrustML (0TML) - Byzantine-resilient federated learning
- **91GB in 0TML/** - Main implementation + experiment results
- Contains significant duplication that accumulated during development
