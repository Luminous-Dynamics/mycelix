# 0TML Notes Archive

High-volume status logs and milestone documents that previously cluttered the
project root are collected here.

## Structure

- `status/` — execution logs, phase completions, sprint notes
- `plans/` — phase roadmaps, deployment plans, strategy adjustments
- `reports/` — testing summaries, validation reports, benchmarking recaps
- `other/` — items awaiting categorisation

Add subfolders if you need finer granularity, but keep the ZeroTrustML root
clean by filing ad-hoc notes under this archive.
