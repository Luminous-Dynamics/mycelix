## E/N/M Classification

Each claim carries an `EpistemicPosition` with three axes (0.0-1.0):

- **Empirical** (E): How verifiable through observation/experiment
- **Normative** (N): How aligned with ethical/social frameworks
- **Mythic/Materiality** (M): What practical/material significance

## Regenerating

```bash
# Generate from live APIs (requires internet)
cd mycelix-knowledge
npx ts-node importers/wikidata-importer.ts
npx ts-node importers/geonames-importer.ts            # curated embedded set
npx ts-node importers/geonames-importer.ts cities15000.txt  # full GeoNames file
npx ts-node importers/openal-importer.ts

# FEVER requires downloading the dataset first
# https://fever.ai/dataset/fever.html
npx ts-node importers/fever-importer.ts /path/to/train.jsonl

# Domain importers (curated datasets, no downloads needed)
npx ts-node importers/pubchem-importer.ts
npx ts-node importers/noaa-importer.ts              # curated; --token TOKEN for API
npx ts-node importers/who-importer.ts                # curated; --api for GHO API
npx ts-node importers/energy-importer.ts             # curated; --api-key KEY for EIA
npx ts-node importers/fao-importer.ts                # curated; --api for FAOSTAT

# Requires downloading MultiFC dataset
npx ts-node importers/multifc-importer.ts /path/to/multifc.tsv

# Requires Google API key with Fact Check Tools API enabled
npx ts-node importers/factcheck-api-importer.ts --key YOUR_API_KEY
```

## Seeding to Conductor

```bash
# Requires running Holochain conductor with mycelix-knowledge hApp
npx ts-node scripts/seed-claims.ts

# Dry run (validate only, no submissions)
npx ts-node scripts/seed-claims.ts --dry-run

# Single file
npx ts-node scripts/seed-claims.ts --file wikidata-core.json

# Custom conductor URL
npx ts-node scripts/seed-claims.ts ws://192.168.1.10:8888
```
