## Detailed Description

### Fight Misinformation with Decentralized Fact-Checking

Mycelix Knowledge empowers you to verify claims and statements directly from your browser. Simply select text, right-click, and get instant fact-check results from a global, decentralized knowledge network.

**Key Features:**

🔍 **Instant Fact-Checking**
Select any text on any webpage and fact-check it with one click. Get verdicts like "True", "Mostly True", "Mixed", or "False" with confidence scores.

📊 **Credibility Scores**
See detailed credibility breakdowns including source diversity, author reputation, and cross-validation with related claims.

🎯 **Epistemic Classification**
Understand whether claims are Empirical (verifiable), Normative (value-based), or Mythic (meaning-making) - helping you think critically about information.

🌐 **Decentralized Network**
Powered by Holochain, not a central server. Your queries contribute to a distributed knowledge graph maintained by the community.

🔒 **Privacy-First**
No tracking. No analytics. Your fact-checks aren't stored on any central server.

**How It Works:**

1. Select text on any webpage
2. Right-click → "Fact Check with Mycelix"
3. View verdict, credibility score, and supporting evidence
4. Contribute your own verified claims

**Perfect For:**
- Researchers verifying sources
- Students checking academic claims
- Journalists investigating statements
- Anyone fighting misinformation

**Requirements:**
- Holochain conductor running locally (or connection to a network node)
- Mycelix Knowledge hApp installed
