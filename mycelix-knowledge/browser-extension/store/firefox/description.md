## Description

### Decentralized Fact-Checking for the Open Web

Mycelix Knowledge brings the power of decentralized fact-checking to your Firefox browser. Verify claims, explore a global knowledge graph, and contribute to collective truth-seeking.

**Core Features:**

🔍 **One-Click Fact-Checking**
Select text → Right-click → Get instant verification against the Mycelix knowledge graph.

📊 **Multi-Factor Credibility**
See how claims score on:
- Source diversity
- Author reputation
- Temporal consistency
- Cross-validation

🎯 **Epistemic Framework**
Every claim is classified as:
- **Empirical** - Verifiable through observation
- **Normative** - Value-based statements
- **Mythic** - Meaning-making narratives

This helps you understand *what kind* of claim you're evaluating.

🌐 **Holochain Powered**
Unlike centralized fact-checkers:
- No single point of failure
- No corporate control over "truth"
- Community-maintained knowledge
- Censorship-resistant

🔒 **Respects Your Privacy**
- No tracking or analytics
- Preferences stored locally only
- Open source and auditable

**Getting Started:**

1. Install this add-on
2. Set up Holochain (see documentation)
3. Start fact-checking!

**Use Cases:**
- Verify news articles
- Check scientific claims
- Research controversial topics
- Contribute your expertise
